<length_optimization>
When translation EXCEEDS character limit, try in this order:

1. Use shorter synonym
   - "Belohnungen" → "Prämien" (German, -4 chars)
   - "requisitos" → "reglas" (Spanish, -3 chars)

2. Remove intensity words if not critical
   - "ZERO BS" → just translate "BS" part
   - "ONLY cash" → "Cash" if context is clear

3. Drop articles (der/die/das/el/la/il)
   - "das Casino" → "Casino"
   - "los premios" → "premios"

4. Use compound words (especially German)
   - "ohne Tricks" → "trickfrei"
   - "ohne Limits" → "limitfrei"

5. Keep English loanwords if shorter
   - "Bargeld" → "Cash" (German, -5 chars)
   - "efectivo" → "cash" (Spanish, -3 chars)

6. Use symbols
   - "and" → "&"
   - "ZERO" → "0" (if maintains tone)

7. Strategic abbreviations (only if natural)
   - "Casino" → "Cas." (RARELY - only if culturally acceptable)

NEVER sacrifice clarity or brand voice for brevity.
Stop when: under limit AND sounds native AND maintains impact.
</length_optimization>
